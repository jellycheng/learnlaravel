1. 官网： https://github.com/ErikDubbelboer/phpRedisAdmin
2. 安装：
	a）下载解压
	b）进去响应目录，composer install 安装依赖代码
	c）把includes/config.sample.inc.php 文件重名为 config.inc.php
	d）config.inc.php文件配置参考：
