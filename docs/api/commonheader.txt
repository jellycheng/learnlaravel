
通用头

app-source： 	
	1.Android app 用A表示
	2. Ios app 用I表示
	3. 触屏H5，浏览器中访问，用H表示
	4. 微信内访问 用W表示


app-v：	当前应用版本号
app-m： Dev：表示是开发版本，Prod：表示正式版本
app-deviceID： 设备唯一标识， android 设备唯一标示算法：由于android机器及rom种类繁多，imei、mac地址、androidId在少部分情况下可能获取不到，
			这个算法逻辑上最大程度上获取唯一标示手机的编码。
			算法依次取:imei、androidid、mac地址, 前面未取到以后面的值替代 

app-channel：	渠道
app-deviceModel	设备型号，比如 [KONKA W850]
app-osVersion	系统版本号
app-type： hf－－－xx房，hft－－－xx拓，gj－－－管家，gjsh－－－管家商户，abaz－－－xx
